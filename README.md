# Portfolio Website

A modern, creative, and professional portfolio website showcasing skills, projects, experience, and personal branding.

## Features

### Design
- **Modern & Clean**: Minimalistic design with vibrant gradients and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Dark Mode Ready**: Prepared for dark mode implementation
- **Smooth Animations**: Scroll animations, hover effects, and dynamic transitions

### Sections
1. **Home**: Hero section with animated introduction and social links
2. **About Me**: Biography with animated statistics counters
3. **Skills**: Categorized skills with animated progress bars
4. **Portfolio**: Filterable project gallery with hover effects
5. **Experience**: Professional timeline with work history
6. **Education**: Academic background and certifications
7. **Testimonials**: Carousel slider with client feedback
8. **Blog**: Latest blog posts (dummy content)
9. **Contact**: Functional contact form with validation
10. **Footer**: Quick links and newsletter subscription

### Interactive Features
- Smooth scrolling navigation
- Active section highlighting in navbar
- Portfolio project filtering
- Testimonials auto-carousel
- Contact form validation
- Scroll-to-top button
- Lazy loading support
- Animated counters and progress bars
- Ripple button effects

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with custom properties, flexbox, and grid
- **JavaScript**: Vanilla JS for interactivity (no dependencies)
- **Font Awesome**: Icon library
- **Google Fonts**: Poppins font family

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic web server (optional, for local development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/shoaibswe45/portfolio-website.git
cd portfolio-website
```

2. Open the website:
   - **Option 1**: Simply open `index.html` in your browser
   - **Option 2**: Use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

3. Visit `http://localhost:8000` in your browser

## Customization

### Personal Information
Edit the content in `index.html` to add your own:
- Name and tagline
- Biography and experience
- Skills and proficiency levels
- Projects and portfolio items
- Education and certifications
- Testimonials
- Contact information

### Styling
Modify `styles.css` to customize:
- Color scheme (edit CSS variables in `:root`)
- Typography and fonts
- Layout and spacing
- Animation speeds

### Functionality
Update `script.js` to modify:
- Animation triggers and timing
- Form submission behavior
- Carousel settings
- Filter functionality
